FEATURED VISUALIZATION:
HYPERSHAPES

We are used to drawing 2D projections of 3D shapes. This process can be extended to n-dimensions by defining a 2D direction for each of the n coordinate axes. This example shows the result for various shapes including the unit simplex, unit cube, and unit sphere.
